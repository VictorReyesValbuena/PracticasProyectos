# ¿Qué es este repositorio?

En este repositorio, subiré las prácticas y proyectos que representen y demuestren diferentes conocimientos adquiridos.

## A - Proyecto 1: 

El objetivo final es crear la función `average_matrix` que recibe una matriz de `float`s y devuelve una nueva matriz 
de las mismas dimensiones, pero con cada punto sustituído por el promedio de sus vecinos y él mismo.
Se trata de *promediar* los valores de la matriz.

## B - Proyecto 2: 

Es una puesta en práctica de *la regresión lineal* por el *método de mínimos cuadrados* utilizando `Python`. Se explorará un dataset, se 
se implementará la *regresión lineal* para luego aplicarla al dataset concreto. Se utilizarán librerías como `matplotlib`, `numpy` o 
`sklearn`.

## C - Proyecto 2:

Sobre un dataframe y utilizando el lenguaje `R`, se hará un trabajo de gestión del dataframe diferentes test de viabilidad y, por último,
intentar hacer una predicción utilizando la *regresión lineal*.
